# ⚡ Quick Setup Card - Strava Integration

## 30-Second Setup

### 1️⃣ Get Strava Credentials (5 min)
- Go: https://www.strava.com/settings/api
- Create New Application
- Copy Client ID & Secret

### 2️⃣ Add to Replit Secrets (1 min)
- Click Secrets icon (🔐)
- Add 2 new secrets:
  - `STRAVA_CLIENT_ID` = your ID
  - `STRAVA_CLIENT_SECRET` = your secret
- *(Redirect URI has default, no 3rd secret needed)*

### 3️⃣ You're Done! ✅
- Backend automatically picks up secrets
- No .env file needed
- No code changes needed
